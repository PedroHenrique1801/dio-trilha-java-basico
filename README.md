# Trilha Java Básico - DIO

Este repositório foi criado para documentar o meu progresso, armazenar as resoluções dos exercícios e desafios de código propostos no Santander Bootcamp 2026 - AI Java Back-end

O objetivo principal é aplicar conceitos de lógica de programação, boas práticas de escrita de código (Clean Code) e conceitos de Programação Orientada a Objetos (POO).

## Estrutura do Repositório

O código fonte está organizado dentro do diretório `src/`, dividido pelas seguintes etapas de aprendizado:

*  fundamentos
*  estrutura-controles
*  classes-encapsulamentos
*  heranca-polimorfismo
*  interface-lambdas
*  collections

## Tecnologias Utilizadas

* **Linguagem:** Java 17+
